ETS2 Telemetry 웹 서버 및 모바일 대시보드에 오신걸 환영합니다!

================
폴더 구조
================

	----------
	| mobile |  미리 컴파일된 안드로이드 앱 (한글화X)
	----------
	| server |  미리 컴파일된 서버 및 모바일 대시보드와 스킨
	----------
	| source |  이 프로젝트의 소스코드
	----------

"server\Ets2Telemetry.exe"만 실행하시면 서버가 시작되며, 사용자의 기기를 HTML5 모바일 대시보드에 연결하십시오!

================
     링크들
================

새 버전과 업데이트 및 도움말 그리고 질의응답은 아래 URL에서 가능합니다.
https://github.com/Funbit/ets2-telemetry-server

한글화 버전 관련 문의는 아래 URL에서 가능합니다.
https://github.com/icaros7/ets2-kor-telemetry-server
